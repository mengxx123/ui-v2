模态框

可使用class来快速改变按钮的颜色，如.btn-primary

参数介绍
选项 默认值 说明
backdrop true 点击对话框周围位置隐藏对话框
keyboard true 按 esc 键关闭对话框
show true 按 esc 键关闭对话框
模态框
方法 说明
$(selector).modal(options); 初始化模态框（初始化之后默认显示模态框）
$(selector).modal('toggle'); 切换模态框
$(selector).modal('show'); 显示模态框
$(selector).modal('hide'); 隐藏模态框
事件
事件 说明
show.bs.modal 当 show 实例方法被调用之后，该事件被触发。
shown.bs.modal 当模态框显示出来之后（同时 CSS 过渡效果也已执行完毕），该事件被触发。
hide.bs.modal 当 hide 实例方法被调用之后，该事件被触发。
hidden.bs.modal 当模态框向用户隐藏之后（同时 CSS 过渡效果也已执行完毕），该事件被触发。
