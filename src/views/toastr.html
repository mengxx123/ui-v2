提示消息（Toast）

Toast 用来向用户显示信息，不同于 Javascript 的 alert， 它不会打断用户的操作，因此用户体验比较好。

常用来显示短时间的提示信息。比如提示用户当前操作的状态反馈等等。

消息
方法 说明
toastr['success']("Hello World"); 显示成功消息。success 可换成info、warning、error
toastr.success("title", "content") 显示成功消息
toastr.clear() 清除所有消息
toastr.clear(toast) 清除某个消息
参数介绍
选项 默认值 说明

closeButton false 显示关闭按钮

hideDuration false 隐藏动画时间（毫秒）
timeOut false 显示时间（毫秒）
extendedTimeOut false 指标离开消息后的显示时间（毫秒）
